Digital Collective Gov Releated Repos

Government-related, open github accounts

A curated collection of github accounts which contain open source code from government related organisation in the UK.

See the collection

Fllowing GDS Service Standard, Point 12 Civil Service produced many open repositories with code relating to the services they provide.

This collection provides a list of github accounts with such code as a starting point for anybody who wants to explore what is there.

Some of the government organisations are listed at government.github.com. Unfortunately, not all accounts created by cross government communities are eligable for inclusion there.

This is work in progress, not a finished product

Currently, the government standards call for that sharing through APIs, but perhaps there is a simpler and more practical option? At the moment this page represents nothing more than exploration of what’s possible and it doesn’t intend to be a full dataset.
